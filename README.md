## 根本目的

AI技術が激変し続ける中で、特定のツール習熟や目先の利益に振り回されることなく繰り返し使える「戦略の型」を確立する。  
その型によって価値提供し、経済的自立を成し遂げる。

## 現在地（2026-04-24〜）

既存SNSの諸問題（主にアテンション・エコノミーと広告モデルによる構造的問題）を再現させない環境としての「Nagi」開発から撤退。  
単一プロダクトへの執着を捨て、**「何を作るか」より「なぜそう判断し、どう検証したか」** を軸にした仮説検証サイクルに移行。  
成果物ではなく判断ログを蓄積する。

## なぜこのアプローチか

- 生成AIにより「作ること」の価値が相対的に低下した
- 差別化できるのは"Why"の部分と、人間固有の文脈・判断・*非合理性*
- AIに（恐らくは短期で）置き換えられるスキルの習得にコストを割くのは失敗戦略

## このリポジトリの構造

- `docs/` — 日付つき判断・思考ログ
- `projects/` — 仮説検証プロジェクトの記録
- `POLICY.md` — 運用ルールとAIへの指示仕様
- `INDEX.md` — 全ドキュメントの索引（自動更新）
