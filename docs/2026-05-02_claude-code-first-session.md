## 気づき・仕様メモ

- `shift+tab`の意味：セッション中の全編集を確認なしで許可するショートカット
- GitHub Actionsのトリガー：`docs/**/*.md` または `projects/**/*.md` の変更時のみ（空commitでは走らない）
- `projects/`はINDEX.mdの管理外（CLAUDE.mdに明記済み）
- セキュリティリスク：OSSへの悪意あるコード混入をAIエージェントが自動インストールするケースが発生中。権限委譲は段階的に絞る方針
