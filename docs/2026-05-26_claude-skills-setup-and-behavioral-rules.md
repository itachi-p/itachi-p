# Claude Skills検証・スキル登録・行動ルール制定

## 判断一覧

---

### 1. Claude Skillsの仕組み確認

**判断：** 検証完了、制約を把握した上で運用継続

**経緯：**
- スキルは無料プランでも使用可能であることを公式ドキュメントで確認
- Chatから使うには ZIPアップロード + コード実行の有効化が前提条件
- プロジェクト内チャットではスキルが動作しないことが判明

**学び：**
スキルの動作環境がプロジェクト内チャットに限定されない——むしろ**プロジェクト外のChatが主戦場**という逆転した前提になる。セットアップコストは低い（ZIP + 設定のみ）が、利用文脈を誤ると全く機能しない。

### 4. 0時以降即就寝のIF-Thenルール制定

**判断：** 採用（本日より実施）

**経緯：**
就労支援スタッフとの面談を経て制定。「意志力で寝る」ではなく「ルールで強制する」構造に切り替え。

**根拠：**
意志力依存の習慣維持は再現性が低い。IF-Thenルールに落とし込むことで、判断コストと失敗率を下げる。TOC的には「そもそも意志の問題として扱わない」がボトルネック解消に近い。

---
