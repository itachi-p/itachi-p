# 「比較」ではなく「協働」が自然に発動する社会は作れるのか

── Podcast『タグログ』#24から広がった思考メモ

このnoteは、ボーダレス・ジャパン田口一成さんがメインスピーカーを努め得るPodcast『タグログ』#24の中盤以降を起点にした思考メモです。

番組内の発言を手がかりにしながら、「おそらくこういう問題意識ではないか」という仮説的な読み替えを行い、その上でSNSや現代社会における比較構造・同質性・自己効力感の問題へと一般化しています。

最終的には、「競争に勝つこと」を前提とした構造から、「協働と知見の再利用（巨人の肩に乗る構造）」へと移行可能か、というシステム設計的な仮説までを整理しています。

1. 「比較し続ける社会」の疲弊

『タグログ』#24では、中盤以降に「比較」の話が出てくる。

上を見て落ち込む。下を見て安心する。どちらも違う。

田口さん自身は、そこをかなり感覚的に語っていた印象がある。「最近の言葉でいう自己効力感なのかな？」と少し迷いながら。

私自身は、その話を聴きながら、

自己効力感

同質性（画一性）

相対評価依存

この3つが強く結びついているように感じた。

2. 日本社会では「安心」が比較に変換されやすい

人は本来、「自分はこれで大丈夫だ」という安心や実感を求めている。

ただ、日本社会ではそれが、

正解側にいること

多数派から外れていないこと

周囲から承認されること

に変換されやすい。

「幸せの形は人それぞれ」と言いながら、

「でも、自分は正しい側にいるよね？」「みんな認めてくれるよね？」

という確認を、常に求め続ける構造。

SNSは、それを極めて効率よく可視化・増幅してしまった。

3. SNSは“原因”というより“増幅器”

SNSだけが悪いわけではない。

むしろ既に存在していた、

比較文化

同調圧力

承認依存

相対評価社会

を、可視化して加速している。

だからSNS上では、

見下し

陣営化

正義の殴り合い

優越確認

が起きやすい。

求めているのは安心や自己効力感なのに、出力だけが歪んでしまう。

4. 「競争に勝つ」が前提の社会構造

さらに考えていくと、この問題はSNSだけでは終わらない。

現代の資本主義社会そのものが、

比較

相対評価

競争

局所最適

を前提に設計されている。

四半期利益。シェア争い。囲い込み。秘密主義。

「自分のところだけ勝てばいい」が、合理的行動として成立してしまう。

もちろん競争そのものを完全否定するつもりはない。

ただ、「協働によって全体が前進する構造」とは、本質的に相性が悪い部分がある。

5. 「巨人の肩に乗る社会」は作れるのか

一方で、『タグログ』#24後半で田口さんが言わんとしていたことは、おそらく、

誰かの発見が再利用される

協力が自然に発動する

知見が社会共有財産として積み上がる

そういう方向性だったのではないか、と私は解釈した。

これは言い換えると、

「競争による局所最適」ではなく、「協働による全体最適」

への転換でもある。

そして、その構造が本当に成立するなら、

「車輪の再発明」を延々と繰り返さなくても済む社会に近づけるかもしれない。

6. まだ結論ではない

もちろん、これはまだ整理途中の仮説に過ぎない。

理想論で終わる可能性もある。

「協働が大事」という話だけなら、昔から何度も繰り返されてきた。

それでも、少なくとも今の社会は、

比較疲れ

承認依存

分断

短期最適化

によるコストを、かなり大きく支払っているように見える。

だからこそ、

「そもそも目指すゴール自体を変えない限り、プロセスも変わらないのでは？」

という問いは、一度立ててみる価値がある気がしている。
